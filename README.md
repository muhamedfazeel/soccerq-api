## Description

Nest framework TypeScript boilerplate repository.

## Installation

```bash
$ npm i
```

## Running the app

```bash
# start
$ npm run start

# dev mode
$ npm run dev

# debug mode
$ npm run debug

# production mode
$ npm run start:prod
```
